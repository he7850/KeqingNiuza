# 刻记牛杂店 流浪大地的乐团精简版

原项目已停止维护，一些功能已不可用。根据最终版项目，精简留下风花琴自动演奏功能，并增加了：

- 曲目文件夹更新后重载
- 半音自动升key
- 演奏时调整速度
- 支持输出到WindsongLyre练琴程序
- 音轨中文名称支持
- 热键调整
